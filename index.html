Components of a Search Engine

Robot

A robot is a program that automatically traverses the hypertext Sturcture of the Web by retreiving a web page document and following the hyperlinks on page. In addition, some robots may address and store the title, meta tag descriptions, and some text on the page.

Database

The collection of information that is so organized so that its contents can be accessed, manged, and updated. The web page that displays the results of your results of you search has information from the Database that is used by the search engine site. Some search engines use some of the content received from other search engines. Such as Yahoo! which gets its primary search results from Google.

Search Engine Optimization

Keywords

Spend time brainstorming terms and phases that people may use when searching for your site. These terms are what descibe your website or business.

Page Title

A descriptive page title which includes your company name or website name will help your site market itself. The page title is often displayed in the SERP and saved when a vistior bookmarks your page. Do not use the exact same title for each page of your website, inculde keywords in the page title that are appropite for the page

Heading Tags

Use structured tags, h1,h2, ect to organize your content. If appropite include some keywords in th text contained within the heading tags. Some search engines will give higher list position if keywords are used in the page title or heading. However, do not spam keywords, meaning repeating the words over within the content of the page. Search engines continue to become more sophisicated and spamming can prevent your page from being listed if it percives you trying to cheat the system.

Description

Write a few sentences about your website or business that makes it speacial and why people would want to visit. The Description should be inviting and intresting so that someone would choose your site.

Description Meta Tag

A self-contained tag that is placed in the head section of a web page and is displayed on the SERP by some search engines, such as Google.

Linking

Verify that all hyperlinks are working. Each page on oyur website should be reachable by the text hyperlink and should be descriptive. Do not use phases such as info or click here. Should use keywords as appropiate.

Images and Multimedia

Search engine robots do not see the text within your images. Configure meaningful alt text for images and inculde relvent keywords.

Valid Code

While it is not required by search engines that your code be validated, well structured and valid code is more easily processed by search engine robots which could help your placement in the search engine.

Content of Value

Your website should contain high-quality, well-organized content that is of value to your visitors.

Additional Techniques for SEO

Map Your Site

An HTML site map is a webpage with a map of the site that contains hierarchial list of hyperlinks to the major pages in your website. this can be helpful for the website visitors and assits search engine robots as they follow hyperlinks on your site. An XML site map provides the following information about each page to a search engine:

  • date that each page was last modified
  • indicator of how frequently each page changes
  • priority level for each page

You can use an online sitemap genertor to create a sitemap file.

Link Popularity

Search engine rate websites based on the number of sites that link to a particulat website and the quality of those sites. Link popularity can determine its order in the SERP.

SMO

Make it easy for your current and potential wedsite vistiors by creating content of value that is sharable. Such as providing links to Twitter, Facebook, and Pinterest.
